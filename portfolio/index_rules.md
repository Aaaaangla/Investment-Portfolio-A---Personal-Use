## **Return - CAGR**

$$
CAGR = \left(\frac{P_{end}}{P_{start}}\right)^{1/n} - 1
$$

* $(P_{start})$ : 期初价格
* $(P_{end})$ : 期末价格
* $n$ : 年数

> [!warning]
> CAGR 不关心中途怎么波动，只看起点和终点。

数学上，它回答的是：
> “如果这笔投资每年都按同一个比例增长，那这个比例是多少？”

| CAGR 区间 | 数学含义   | 投资评价    |
| ------- | ------ | ------- |
| < 2%    | 接近通胀   | ❌ 不值得   |
| 2%–5%   | 勉强跑赢通胀 | ⚠️ 防守   |
| 5%–8%   | 稳定资产   | 🟡 合格   |
| 8%–12%  | 长期优秀   | 🟢 很好   |
| 12%–18% | 高质量资产  | 🔵 极好   |
| > 18%   | 要警惕    | ⚠️ 不可持续 |

---

**1年CAGR和5年CAGR都要计算**

| 情况                | 数学解释 |
| ----------------- | ---- |
| 1Y CAGR ≈ 5Y CAGR | 正常   |
| 1Y ≫ 5Y           | 可能过热 |
| 1Y ≪ 5Y           | 可能回调 |

## **Volatility波动率**

### 1️⃣ Daily Return

$$
r_t = \frac{P_t - P_{t-1}}{P_{t-1}}
$$

---

### 2️⃣ Daily Volatility

$$
\sigma \approx \sqrt{\frac{1}{N-1}\sum r_t^2}
$$

其中：

* $$( r_t )$$：第 (t) 天的收益率
* $$( \bar{r} )$$：平均日收益率
* $$( N )$$：样本天数

---

### 3️⃣ Annualised Volatility（**投资中真正用的**）

$$
\sigma_{annual} = \sigma_{daily} \times \sqrt{252}
$$

> 252 = 一年大约的交易日数量

---

### Volatility 在投资里“真正代表什么”

> **Volatility ≠ 下跌概率
> Volatility = 回报的离散程度**

也就是说：

* 高波动：可能暴涨，也可能暴跌
* 低波动：结果更可预测

---

### Volatility Standard

| 年化波动率         | 数学含义  | 投资评价    |
| ------------- | ----- | ------- |
| **< 10%**     | 非常稳定  | 🟢 极低风险 |
| **10% – 15%** | 稳定    | 🟢 防守型  |
| **15% – 20%** | 可接受   | 🟡 均衡型  |
| **20% – 25%** | 偏高    | 🟠 成长型  |
| **25% – 35%** | 高波动   | 🔴 高风险  |
| **> 35%**     | 极端不稳定 | 🔴 投机   |

## Beta 系统性风险

> **Beta 衡量的是：
> 一只资产相对于“整个市场”的波动敏感度，而不是它自己有多波动。**

* Volatility：**自己抖不抖**
* **Beta：跟着市场抖得多不多**

$$
\beta_i = \frac{\mathrm{Cov}(r_i,; r_m)}{\mathrm{Var}(r_m)}
$$

其中：

* $$(r_i)$$：资产 (i) 的收益率（如 MSFT）
* $$(r_m)$$：**市场组合**的收益率（通常是 S&P 500）
* $$Cov$$：协方差
* $$Var$$：市场收益的方差

👉 **Beta = 回归斜率**

$$
\beta = \frac{\text{资产与市场的共振强度}}{\text{市场自身的波动强度}}
$$

**工程上面用**：

$$
\beta_i \approx
\frac{\sum_{t=1}^{N} r_{i,t}, r_{m,t}}
{\sum_{t=1}^{N} r_{m,t}^2}
$$

---

### Beta Standard & Meaaning

| Beta  | 数学含义    | 投资语言 |
| ----- | ------- | ---- |
| β = 1 | 与市场同涨同跌 | 市场风险 |
| β > 1 | 比市场更敏感  | 放大器  |
| β < 1 | 比市场稳定   | 防守   |
| β = 0 | 与市场无关   | 非相关  |
| β < 0 | 反向运动    | 对冲   |

---

### Beta 的行业标准区间

#### 📊 Beta Risk Classification（行业共识）

| Beta 范围   | 系统性风险 | 投资风格    |
| --------- | ----- | ------- |
| < 0.5     | 极低    | 防守 / 类债 |
| 0.5 – 0.8 | 偏低    | 稳定型     |
| 0.8 – 1.2 | 中性    | 市场型     |
| 1.2 – 1.5 | 偏高    | 成长型     |
| 1.5 – 2.0 | 高     | 高风险     |
| > 2.0     | 极高    | 投机      |

👉 **这是和你 Volatility Standard 同等级的“风险语言”**

> [!warning]
> ⚠️ Beta **不是物理常数**：
>   * 会随时间变
>   *  会随市场 regime 变
>   *  对危机期不稳定

所以行业做法是：

> **3–5 年滚动 Beta + 定期更新**
